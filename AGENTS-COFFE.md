# AGENTS-COFFE.md

## Tujuan Project

Bangun aplikasi **Coffee Order / Coffee POS** menggunakan:

- Frontend: **Svelte / SvelteKit**
- Backend: **Golang**
- Database: sesuaikan backend, direkomendasikan SQLite untuk awal lalu bisa pindah PostgreSQL
- Preview UI mengikuti file gambar di folder project:
  - `dashboard_utama.jpeg`
  - `history-pesanan.jpeg`
  - `menu-pesanan.jpeg`
  - `menu-profile.jpeg`

Project ini digunakan untuk kelola coffee shop:

- Dashboard utama
- Menu pesanan
- History pesanan
- Profile / pengaturan
- pencarian daftar outlet
